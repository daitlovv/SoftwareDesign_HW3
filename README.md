# Система антиплагиата

Микросервисная система для проверки студенческих работ.

## Состав системы
- **Gateway** (порт 6000) - основной API
- **File Storage** (порт 5001) - хранение файлов
- **File Analysis** (порт 5003) - анализ на плагиат

## Запуск
```bash
docker-compose up -d
```

## Основные endpoints
- `POST /works/upload` - загрузить работу
- `GET /works/{workId}/reports` - получить отчеты
- `GET /health` - проверка работы

## Алгоритм проверки
1. Извлекаем слова из текста
2. Удаляем стоп-слова и короткие слова
3. Считаем процент совпадения уникальных слов
4. Если сходство >70% - плагиат

Формула: `сходство = (общие_слова / max(слова_в_работе1, слова_в_работе2)) * 100%`

## Swagger
- Gateway: http://localhost:6000/swagger
- File Storage: http://localhost:5001/swagger
- File Analysis: http://localhost:5003/swagger
